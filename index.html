iCal Filter Configurator

This page will help you create the configuration file needed to filter your calendar. The configuration will be saved to this repository.

Step 1: Enter your iCal Link

Step 2: Choose Events to HIDE

Check the box next to any event type you want to remove from your calendar.

Step 3: Generate Configuration File

This will take you to GitHub to save a `config.json` file to your repository. Simply scroll down and click the green "Commit new file" button on that page.
